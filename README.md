# IS4302 Project

IS4302 Blockchain and Distributed Ledger Technologies

## Description

A Blockchain-based Carbon Credit Ecosystem that facilitates the exchange of carbon credits.

## Project Setup

### Installing node modules

```
npm install
cd vapp/
npm install
```

### Start vue app

```
cd vapp/
npm run serv 
```

### Launch truffle

```
truffle develop
>>> migrate
```
